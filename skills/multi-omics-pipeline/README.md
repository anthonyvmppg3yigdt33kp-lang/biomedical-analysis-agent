# multi-omics-pipeline skill

Design and audit multi-omics integration while preserving sample identity,
modality-specific preprocessing, leakage-safe validation, method alternatives,
and conclusion boundaries.

Read `SKILL.md` before use. A joint embedding or latent factor is exploratory
unless replicate structure, sensitivity, and external/held-out validation
support a stronger claim.

The bundled R example is syntax-checked by the repository R smoke workflow; it
is not data-verified merely because it parses. Run the complete repository test
suite from the source repository root before distribution.
